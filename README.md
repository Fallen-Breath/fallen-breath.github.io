# personal-homepage

Framework: [Hugo](https://gohugo.io/)

Theme: [lynx](https://github.com/jpanther/lynx)

Deployment url: https://fallenbreath.me

## build

1. Install hugo
2. Build with hugo

    ```bash
    hugo           # regular build
    hugo --minify  # build and minify
    ```
3. Stuffs are now in the `./public` directory
